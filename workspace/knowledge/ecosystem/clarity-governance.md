# Clarity Protocol

Clarity is a governance platform for Cardano enabling on-chain voting.

## Key Features

- **Proposal Tracking**: View and analyze governance proposals
- **Voting History**: Track votes by address or proposal
- **DRep Support**: Delegated Representative information

## Tools Available

- `clarity_get_proposals` - List active/recent proposals
- `cardano_drep_info` - DRep details and voting power
