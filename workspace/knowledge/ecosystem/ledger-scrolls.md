# BEACN Ledger-Scrolls

Ledger-Scrolls is immutable document storage on Cardano tx metadata.

## Key Features

- **Immutable Storage**: Once written, cannot be modified
- **On-chain Verification**: Anyone can verify content
- **Content Hashing**: SHA-256 ensures integrity

## Tools Available

- `scrolls_read` - Read a scroll by transaction hash
- `scrolls_prepare_write` - Prepare metadata for new entry
