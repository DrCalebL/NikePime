# NABU VPN - Decentralized VPN Service

## Overview

NABU provides decentralized VPN services on Cardano, enabling privacy-preserving internet access through a distributed node network.

## Features

- **Decentralized Nodes**: No central point of failure
- **Global Coverage**: Nodes across multiple regions
- **Privacy First**: No logging, no tracking
- **Node Incentives**: Node operators earn rewards
- **Transparent Stats**: Public performance metrics

## How It Works

1. Users connect to the NABU network
2. Traffic is routed through decentralized nodes
3. Node operators provide bandwidth and earn rewards
4. Payment handled via Cardano native tokens

## API Integration

NABU provides a REST API:

- Base URL: `https://api.b7s.services/v1`
- Authentication: API key (optional)
- Documentation: https://api.b7s.services/swagger/index.html

## Available Tools

| Tool                  | Description                |
| --------------------- | -------------------------- |
| `nabu_get_nodes`      | List available VPN nodes   |
| `nabu_get_node_stats` | Get node performance stats |
| `nabu_check_status`   | Check service status       |

## Node Regions

- North America (US-East, US-West, Canada)
- Europe (EU-West, EU-Central)
- Asia Pacific (Singapore, Japan)
