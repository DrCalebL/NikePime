# Surge Cardano

Surge is a decentralized exchange (DEX) on Cardano using AMM technology.

## Key Features

- **Liquidity Pools**: Paired token pools enabling swaps
- **Automated Pricing**: Constant product formula (x\*y=k)
- **LP Tokens**: Represent share of liquidity pool

## Tools Available

- `surge_get_pools` - List liquidity pools with TVL and APY
- `surge_get_quote` - Get swap quote with price impact
- `surge_get_price` - Check token prices in ADA
