# Liqwid Finance

Liqwid is the leading lending and borrowing protocol on Cardano.

## Key Features

- **Lending Markets**: Supply ADA, DJED, iUSD to earn interest
- **Borrowing**: Borrow against supplied collateral
- **Health Factor**: Position health tracked (>1.0 is safe)

## Tools Available

- `liqwid_get_markets` - View all lending markets and rates
- `liqwid_get_position` - Check wallet supplied/borrowed positions
- `liqwid_estimate` - Calculate potential returns or costs
